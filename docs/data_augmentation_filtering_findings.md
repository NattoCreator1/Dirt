# 数据增强筛选研究结论

**日期**: 2026-02-24
**实验**: SD合成数据作为WoodScape训练数据的增强

---

## 1. 研究背景

### 1.1 目标
使用Stable Diffusion生成的8960张合成污渍图像作为训练数据增强，提升模型在真实场景下的泛化能力。

### 1.2 Baseline模型
- **架构**: BaselineDualHead (ResNet18)
- **配置**: S_full_wgap_alpha50
- **性能**: Test Glob MAE=0.0192, bias=-0.002, ρ=0.9919

---

## 2. 手动筛选的失败 (680张)

### 2.1 筛选方法
人工肉眼筛选，偏向选择污渍明显（高S值）的样本。

### 2.2 筛选结果

| 指标 | 值 |
|------|-----|
| 样本数 | 680 |
| S均值 | **0.573** |
| S范围 | [0.020, 1.000] |

### 2.3 质量问题

**Baseline模型预测验证**：
- 误差均值: **0.3753**
- 偏置: **-0.3221** (baseline严重低估)
- 高误差占比 (error>0.1): **85.3%**

**误差随S值恶化**：
```
S区间     平均误差    样本数
0-0.2     0.1928      79
0.2-0.4   0.0825     110  ← 唯一较为准确
0.4-0.6   0.2404     153
0.6-0.8   0.4736     179
0.8-1.0   0.6876     159
```

### 2.4 对Mixed模型的影响

使用680张手动筛选数据混合训练后：
- **Test Glob MAE**: 0.0192 → 0.0996 (5倍恶化)
- **Bias**: -0.002 → **+0.099** (系统性高估)
- **Ext ρ**: 0.7177 → 0.171 (跨域崩塌)

**根本原因**：NPZ标签在高S区间严重偏离baseline预测，模型学习到整体提高预测值以拟合高S标签，这种"补偿"策略在真实数据上导致系统性高估。

## 3. Baseline模型筛选 (989张)

### 3.1 筛选方法
1. 对8960张SD图像使用baseline模型预测 S_hat
2. 计算 |S_hat - S_gt| 误差
3. 筛选条件：误差 < 0.05

### 3.2 筛选结果

| 指标 | 全部8960张 | 筛选后989张 |
|------|-----------|------------|
| 样本数 | 8960 | 989 (11%) |
| S均值 | 0.396 | **0.267** |
| S范围 | [0.003, 1.000] | [0.064, 0.458] |
| 误差均值 | 0.1024 | **0.0245** |
| 偏置 | -0.2212 | **-0.0055** |

### 3.3 质量验证

| 分位数 | 全部8960 | 筛选989 |
|--------|---------|--------|
| 25% | 0.0546 | 0.0128 |
| 50% | 0.0893 | 0.0246 |
| 75% | 0.1395 | 0.0367 |
| 90% | 0.2056 | 0.0440 |

**高误差样本占比 (error>0.1)**：
- 全部8960: 29.2%
- 筛选989: **0.0%**

---

## 4. 核心发现

### 4.1 SD数据标签质量问题

1. **系统性偏移**：SD数据的NPZ标签整体高于baseline对相似内容的预测（偏置 -0.2212）
2. **高S区间标签失真**：S值越高，标签越不准确
3. **仅11%样本可用**：只有989张（误差<0.05）的标签与baseline预测一致

### 4.2 手动筛选的偏差

1. **视觉偏差**：人眼倾向于选择污渍明显的样本（高S值）
2. **忽视标签质量**：忽略了NPZ标签在高S区间的失真问题
3. **结果**：85.3%的筛选样本标签不可靠

### 4.3 Baseline筛选的优势

1. **客观量化**：基于模型预测误差，无主观偏差
2. **质量保证**：筛选后样本误差均值仅0.0245
3. **分布合理**：S均值(0.267)与真实数据更接近

---

## 5. 数据筛选原则

### 原则1: 误差阈值筛选
```
保留 |S_hat_baseline - S_gt| < threshold 的样本
推荐 threshold = 0.05
```

### 原则2: 避免高S区间偏好
- 高S区间 (>0.6) 标签失真严重
- 筛选应关注标签质量，而非视觉冲击力

### 原则3: 保持分布一致性
- 增强数据的S分布应与真实数据相近
- WoodScape S均值 ≈ 0.40
- Baseline筛选 S均值 = 0.267 (合理范围)

---

## 6. 建议

### 6.1 使用Baseline筛选的989张样本
- 标签质量可靠（误差<0.05）
- 分布合理，不会引入偏置
- 建议混合比例：WoodScape(4000) : SD(989) ≈ 4:1

### 6.2 避免使用高S值SD样本
- S > 0.6 的样本标签失真严重
- 如需使用，需重新标注或调整标签

### 6.3 未来数据生成改进
1. **标签生成优化**：检查高S区间的标签生成逻辑
2. **质量控制**：在生成后进行baseline模型筛选
3. **迭代优化**：使用筛选后的高质量样本重新训练LoRA模型

## 8. 关键教训

1. **直觉不可靠**：人工筛选容易陷入视觉偏差，忽视标签质量问题
2. **模型验证必要**：使用已知可靠的模型验证合成数据标签质量
3. **误差阈值筛选**：简单有效的质量控制方法
4. **分布一致性重要**：增强数据的特征分布应与真实数据相近
