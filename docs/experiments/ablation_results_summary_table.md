## 关键结论

1. **S_full_wgap_alpha50** 在 External Test 上达到 **ρ=0.7177**，仅比最优值 (0.7186) 低 0.0009
2. **S_full_wgap_alpha50** 在 External Test 上 **MAE=0.7394, RMSE=0.9065**，均为所有模型中最低
3. 这证明了**两权重系统**概念：通过 w_gap_strong + alpha50 混合调整，可以在保持完整模型结构的同时获得接近最优的性能
4. 不应放弃 Spatial 和 Dominance 组件，它们提供的额外信息是有价值的

## 符号说明

- **WS**: WoodScape Test Set
- **Ext**: External Test Set
- **ρ**: Spearman 等级相关系数
- **MAE**: 平均绝对误差
- **RMSE**: 均方根误差
- **Δρ**: ρ(S_hat) - ρ(S_agg)，双头架构的一致性指标
