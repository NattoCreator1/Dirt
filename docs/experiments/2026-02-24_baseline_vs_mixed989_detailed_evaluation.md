## 五、综合分析

### 5.1 性能对比总结

#### Test_Real (同域，强标注)

| 维度 | Baseline | Mixed (989) | 优胜者 |
|------|----------|-------------|--------|
| Global MAE | 0.01916 | 0.01882 | Mixed |
| Tile MAE | 0.04158 | 0.04301 | Baseline |
| 相关性 ρ | 0.99186 | 0.99141 | 相当 |
| 偏置 bias | -0.0021 | +0.0006 | Mixed |

**结论**: Mixed (989) 在主要指标 (Global MAE) 上略优于 Baseline。

#### Test_Ext (跨域，弱标注)

| 维度 | Baseline | Mixed (989) | 优胜者 |
|------|----------|-------------|--------|
| Spearman ρ | 0.7177 | 0.7097 | Baseline |
| 单调性 | ✓ | ✓ | 相当 |
| 预测范围 | [0.316, 0.746] | [0.304, 0.645] | Baseline |

**结论**: Mixed (989) 的跨域性能略低于 Baseline，但差距很小 (Δρ = -0.008)。

### 5.2 关键发现

1. **Mixed (989) 在同域上略有改进**
   - Global MAE 降低 0.00034
   - 无系统性偏置
   - Level Accuracy 提升

2. **Mixed (989) 在跨域上基本保持**
   - Spearman ρ 仅下降 0.008
   - 单调性保持
   - 预测更保守但仍有效

3. **预测模式的差异**
   - Mixed 在 50.9% 的样本上更好
   - 改进和退化的样本数量相当
   - 整体误差分布 Mixed 更优

### 5.3 结论

**Mixed (989) 模型成功验证了 Baseline 筛选的 SD 数据质量**:

1. ✅ **同域性能**: Global MAE 略有提升
2. ✅ **跨域性能**: Spearman ρ 基本保持
3. ✅ **无副作用**: 无系统性偏置，无性能崩塌
4. ✅ **数据增强有效**: 989 张高质量 SD 数据提供了有益的补充

**与手动筛选版本 (680) 的对比**:

| 指标 | 手动筛选 (680) | Baseline筛选 (989) |
|------|---------------|-------------------|
| Test MAE | 0.0996 | **0.0188** |
| Test ρ | 0.972 | **0.991** |
| Ext ρ | 0.171 | **0.710** |
| Bias | +0.099 | **~0** |

**数据质量是关键**: Baseline 筛选的高质量 SD 数据 (误差<0.05) 比手动筛选的不可靠数据 (85.3%误差>0.1) 更有效。
