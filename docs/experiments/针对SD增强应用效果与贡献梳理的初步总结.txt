你现在得到的现象非常“合理”：*在 WoodScape(\rightarrow)External 的域差存在时*，用“同域 mask + 同域 LoRA”的 SD 合成数据做混合训练，往往更容易带来“*同域不降、跨域不升甚至轻微下降*”的结果；你这里 *Spearman (\rho) 从 (0.7177) 到 (0.7097)*、且 (\hat{S}) 在所有等级整体下移，正好对应了这种机制。

下面按“现象解释 (\rightarrow) 为什么你当前筛选策略很可能限制了增益 (\rightarrow) 如何改造以更有机会提升跨域 (\rightarrow) 如果跨域提升很难，SD 增强在论文主线还能贡献什么”给出严谨梳理。

---

## 1) 你当前结果在统计与机制上意味着什么

### 1.1 (\rho) 降了 (0.008) 的含义

(\Delta\rho= -0.008) 属于很小的差异。是否“真实变差”，需要对 (\rho) 做 bootstrap 置信区间（对 External 样本重采样多次计算 (\rho)），否则很可能落在随机波动范围内。
但你给出的 *等级均值整体下移* 与 *等级分离度在 Level1–4 明显变小* 是更强的结构性信号，说明 Mixed 模型出现了“*输出尺度压缩/保守化*”。

### 1.2 Mixed 模型更保守、分离度下降的典型原因

你给出的表格中，Mixed 的 (\hat{S}) 在 Level 1–5 全部更低，且 Level1→4 的间隔变小，表现为“曲线变平”。这通常对应以下两类根因（两者可叠加）：

1. *合成样本的“视觉严重度”低于其标签严重度*
   你合成时使用的是 WoodScape 的 mask（其形状边界偏块状、缺少真实薄层的自然渐变），且 SD 生成的纹理/光学效应如果不足，会出现“看起来没那么严重，但标签 (S) 仍然很高”。模型在训练中会被迫学到一个更保守的映射：同样的视觉证据对应更低的 (\hat{S})，从而在 External 上整体下移、并压缩等级间隔。

2. *筛选策略产生“确认偏差”，把增强数据变成“重复同域易样本”*
   你用 baseline 推理后筛掉“与标签偏差大”的合成图，最终保留 989 张。直觉上这是“保真”，但它会把增强数据收缩到“baseline 已经会的那一小块分布”——也就是更像 WoodScape、对 baseline 来说更容易、信息增量更低的样本。
   这种数据加入训练后，更像是一种“同域正则/蒸馏”，对跨域提升帮助很有限，甚至可能把模型推向“更保守、更不敢给高分”的校准状态（尤其当保留样本更偏轻污或纹理弱时）。

---

## 2) 为什么“仅基于 WoodScape 做增强”很难显著提升 External

你的判断是对的：当 *mask 形状、相机成像链、噪声模型、颜色响应、动态范围、压缩伪影* 都与 External 不一致时，即便 SD 的纹理更真实，仍可能只是在 WoodScape 域内“更丰富”，对 External 的关键差异维度覆盖不够。

更具体地说，你现在 SD 增强的“可变因素”主要是：

* clean 帧随机组合；
* mask 来自 WoodScape；
* LoRA 学到的脏污外观来自 WoodScape。

这意味着合成数据的分布很可能仍然集中在 WoodScape 的成像统计上，而 External 的域差（例如曝光/白平衡/噪声/模糊/压缩/镜头结构/遮挡形态）没有被充分随机化或覆盖。

## 3) 当前 SD 增强工作仍然可能有的“论文贡献点”

即使 External 的 (\rho) 没有提升，你的 SD 增强工作仍然可以对论文主线产生价值，但需要把“贡献叙事”定位得更准确：

### 3.1 贡献点 A：可控合成 + 质量筛选链路本身是方法贡献

你已经实现了：

* 二阶段：LoRA（外观）+ Inpainting（空间约束）；
* 大规模生成（8960）；
* 通过模型一致性筛选出高可信子集（989）。
  这是一条完整的 *“生成—质控—入训”闭环*。在很多论文里，这足以作为“可复现的数据增强系统”贡献，哪怕跨域增益有限，也能证明：增强 *不会破坏同域性能*、并能维持 External 的单调性。

### 3.2 贡献点 B：负结果也可以服务主线：证明“跨域瓶颈主要来自域统计而非 mask 覆盖”

你当前观察到：WoodScape test 持平、External 单调性仍 True，但 (\rho) 略降、整体更保守。
这实际上在支持一个很有价值的结论：**只改变“脏污外观细节”而不显式覆盖“成像域差”时，跨域提升非常有限**。
把它写成讨论（Discussion）中的“限制与启示”，反而能提升论文可信度。

### 3.3 贡献点 C：SD 增强可用于“尾部覆盖/压力测试/下游关联”而非直接追 (\rho)

你现在 Mixed 在 Level4→5 的间隔反而更大（分离度更强），这提示 SD 增强可能更适合：

* 强化极端重污样本的分离；
* 构造可控 stress test（不同透明度、不同形态、不同光照）；
  这些内容与 *Severity Score*、Aggregator 一致性、以及下游车道线退化规律验证，是高度一致的。

## 4) 若目标仍是提升 External 泛化，改进点应聚焦在“覆盖域差的可变因素”

你目前增强的瓶颈不是“有没有 SD”，而是 *合成分布是否覆盖 External 的差异维度*。在保持数据隔离（不使用 External）前提下，最有效的方向是引入“相机域随机化 + 形态随机化”，并调整筛选逻辑避免确认偏差。

### 4.1 生成侧改进：把外观差异扩展到“成像链”层面

在合成完脏污层后，对整图或局部做“相机域随机化”（不需要外部数据）：

* 颜色/曝光：gamma、色温偏移、白平衡漂移、局部过曝/欠曝；
* 噪声：shot noise + read noise（强度随机）、色噪；
* 模糊：轻微 defocus + motion blur（方向随机）；
* 压缩：JPEG 质量随机、色度子采样伪影；
* 镜头效应：vignetting、轻微 chromatic aberration；
* 分辨率链：随机下采样再上采样（模拟 ISP / 编码链）。

这些操作的作用是让网络学习到“与传感器/ISP 相关的稳健特征”，比只改变污渍纹理更可能改善跨域。

### 4.2 mask/alpha 形态改进：从“多边形块”变成“自然薄层”

即使 support 区域仍来自 gtLabels，也应把硬边界变成连续 (\mathbf{A}\in[0,1]) 的 alpha，并加入内部结构调制：
[
\mathbf{A}_{final}=\text{feather}(M)\odot \mathbf{N}
]
其中 (\mathbf{N}) 可由多尺度噪声+方向性 streak 生成。
这能显著降低“贴纸感”，并缓解“视觉严重度低于标签严重度”的问题。

### 4.3 筛选策略改进：不要用单一 baseline 偏差阈值做唯一筛选

你当前筛选“偏差小于阈值”的样本，容易把增强数据变成“重复同域易样本”。更推荐改成 *多目标筛选*：

* *硬约束*（必要条件）：背景污染率低（spill 小）、mask adherence 好、无明显结构破坏；
* *软约束*（保留多样性）：在满足硬约束前提下，按类别/严重度分层抽样，保证 trans/semi/opaque 与 sev 分布平衡；
* *避免确认偏差*：保留一部分“偏差中等但质量过关”的样本，尤其是高严重度/透明薄层类，因为它们才可能提供新信息。

一个更稳的筛选目标是“几何与物理一致性”，而不是“当前模型已经预测得准”。

### 4.4 训练策略改进：防止尺度压缩与过度保守

你 Mixed 模型整体更低，说明训练过程中映射发生了漂移。两种低成本修正：

1. *混合训练权重/课程*：先用较低权重引入合成数据（或先短暂预训练再在真实数据上微调），避免合成样本主导标尺。
2. *加入结构一致性约束*：在 Mixed 训练中启用 (\mu>0) 的一致性损失 (\mathcal{L}_{cons}=|\hat{S}-\tilde{S}|)，可一定程度抑制全局头标尺漂移（尤其当合成外观弱导致全局回归不稳定时）。

## 5) 你现在的 SD 增强是否“必须”做跨域提升？

从审稿角度：*不必须*。必须的是“贡献边界清晰、证据链闭环”。你可以把 SD 增强模块写成以下两种更稳的定位之一：

### 定位 1：*增强不破坏性能 + 提供可控数据用于验证 Severity/Aggregator 与下游退化*

证据：WoodScape test 持平，External 单调性保持 True，说明增强不会破坏跨域排序结构；再用可控合成做下游车道线退化相关性分析，证明 Severity 标尺的功能意义。

### 定位 2：*揭示跨域瓶颈与改进方向*

证据：在 mask 与外观均来自 WoodScape 的情况下，External 提升有限甚至略降，说明跨域增益需要覆盖成像链域差（ISP/噪声/曝光/压缩等）；进而提出并验证“域随机化 + 形态软化”的改进。

这两种定位都能让 SD 增强成为论文主线的“可解释模块”，而不是必须交付一个显著提升的 (\rho)。

## 6) 建议你补充的“严谨证据面板”（对应你目前的结果）

为了把“(\rho) 小幅下降但单调性保持、整体更保守”解释得严谨，建议在现有表格之外补三类证据：

1. *Spearman 的 bootstrap 置信区间*（判断 (\Delta\rho=-0.008) 是否显著）；
2. *尺度压缩证据*：绘制 External 上 (\hat{S}) 的分布（baseline vs mixed）与等级均值的斜率变化；
3. *分桶误差或分桶分离度曲线*：特别关注 Level1–4 间隔变小这一点，给出可视化证明。

综合判断：你现在的增强流程已经证明“系统可运行且不会破坏主任务”；External (\rho) 没升反降的主要原因更可能是“合成数据带来的标尺漂移 + 确认偏差筛选导致信息增量不足”，而不是“SD 增强本身没价值”。将改进聚焦在 *成像链域随机化 + alpha 形态软化 + 去确认偏差筛选 + 训练标尺约束*，才更有机会把 SD 增强转化为可解释的跨域增益。
