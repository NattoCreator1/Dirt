## 三、问题分析

### 3.1 根本原因

1. **SD生成偏差**: Stable Diffusion生成的脏污图像偏向中等严重度
2. **标签生成偏差**: NPZ文件中的S值计算方法可能与WoodScape不一致
3. **筛选偏差**: 保守渐进式筛选倾向于保留"保守"样本，即低S值样本

### 3.2 影响评估

根据实验结果 (`2026-02-25_three_models_comprehensive_comparison_baseline_989_1260.md`):

| 模型 | Test_Real MAE | Test_Real Bias | Test_Ext ρ |
|------|---------------|----------------|------------|
| Baseline | 0.0192 | +0.002 | 0.718 |
| Mixed 1260 | 0.0187 | -0.093 | 0.670 |

**观察到的现象**:
- Mixed模型系统性负偏差 (Test_Real bias=-0.09, Test_Ext bias=-0.20~-0.30)
- 等级区分度下降 (Test_Ext ρ从0.718降至0.670)
- Level 4预测下移 (均值从0.68降至0.44)

**与分布差异的关系**:
- SD数据集中在中低S值区间 [0.2, 0.4]
- 模型过度学习中低严重度特征
- 高严重度预测能力下降

## 四、校准建议

### 4.1 校准策略

基于诊断结果，建议采用**分桶Importance Sampling (D-only)**:

**权重计算**:
```python
# 目标分布: WoodScape
# 当前分布: SD
# 权重 = WoodScape频数 / SD频数

weights[bin_0.0_0.1] = 299 / 14 = 21.36  # 大幅提高
weights[bin_0.1_0.2] = 534 / 279 = 1.91
weights[bin_0.2_0.3] = 363 / 309 = 1.17
weights[bin_0.3_0.4] = 466 / 577 = 0.81  # 降低
weights[bin_0.4_0.5] = 424 / 80 = 5.30   # 大幅提高
weights[bin_0.5_0.6] = 290 / 1 = 290.00 # 极大提高
weights[bin_0.6+]   = inf → 用0.5区间权重
```

### 4.2 实施步骤

**阶段1: D-only (Importance Sampling)**
1. 实现TwoLayerImportanceSampler
2. 计算分桶权重
3. 训练Mixed(8960, D-only)
4. 评估: Test_Real MAE≤0.019, Test_Ext ρ≥0.70

**阶段2: A+-only (Consistency Gating)** (如需要)
1. 使用Baseline作为teacher
2. 实现soft gating机制
3. 仅对SD样本应用gate
4. 评估是否进一步提升
