# 合成图像筛选工具使用指南

## 功能特性

- 🌐 **Web界面**: 响应式设计，支持大屏浏览
- ⚡ **快捷键操作**: 高效筛选（A=接受, R=拒绝, 方向键切换）
- 📊 **实时统计**: 显示浏览进度和筛选结果
- 🔍 **智能筛选**: 按状态、spill_rate筛选
- 💾 **自动保存**: 筛选决定自动保存到本地
- 📤 **结果导出**: 导出筛选结果CSV

## 快速开始

### 1. 启动筛选工具

```bash
cd /home/yf/soiling_project/sd_scripts/lora
python filter_synthetic_images.py --data_dir /home/yf/soiling_project/synthetic_soiling/batch_896f_10masks
```

### 2. 访问Web界面

打开浏览器访问: `http://127.0.0.1:5000`

### 3. 开始筛选

1. **浏览模式**: 点击任意图像打开大图预览
2. **筛选操作**:
   - 按 `A` 或点击 "✓ 接受" - 标记为高质量
   - 按 `R` 或点击 "✗ 拒绝" - 标记为低质量
   - 按 `←/→` 切换上一张/下一张
   - 按 `Esc` 关闭预览

## 筛选标准建议

### ✓ 接受 (Accept)

- `spill_rate < 0.3` (背景污染少)
- 脏污纹理清晰自然
- 没有明显的杂色/纹理干扰
- 视觉上拟真的镜头脏污效果

### ✗ 拒绝 (Reject)

- `spill_rate >= 0.5` (背景污染严重)
- 脏污纹理混杂其他图案
- 完全看不出脏污效果
- 产生明显的不自然变形

### ? 边界案例

- `spill_rate 0.3-0.5`: 仔细判断，倾向于接受
- 轻微纹理干扰: 如果脏污效果明显，可以接受

## 界面功能

### 侧边栏
- **统计面板**: 显示总图像数、已浏览/接受/拒绝数量
- **进度条**: 可视化浏览进度
- **筛选器**:
  - 状态筛选 (全部/待浏览/已接受/已拒绝)
  - Spill Rate阈值滑块
  - 排序选项
- **导出按钮**: 导出筛选结果

### 主区域
- **工具栏**: 搜索框、显示信息
- **图像网格**: 缩略图预览，带状态标识
  - 绿色边框 = 已接受
  - 红色边框 = 已拒绝
  - 黄色边框 = 已浏览

### 大图预览
- 显示当前图像的完整质量指标
- Caption信息
- 快捷键提示

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| `A` | 接受当前图像 |
| `R` | 拒绝当前图像 |
| `←` | 上一张图像 |
| `→` | 下一张图像 |
| `Esc` | 关闭大图预览 |

## 导出结果

点击侧边栏的 "💾 导出结果" 按钮，会下载一个CSV文件，包含:

| 列名 | 说明 |
|------|------|
| `output_filename` | 图像文件名 |
| `filter_status` | 筛选状态 (accepted/rejected) |
| `spill_rate` | 背景污染率 |
| `mask_coverage` | Mask覆盖率 |
| `mask_dominant_class` | 脏污类别 |
| `caption` | 生成使用的caption |

## 高效筛选技巧

### 1. 分批筛选
- 每次筛选100-200张，避免疲劳
- 使用"待浏览"筛选器继续未完成的工作

### 2. 先筛选明显案例
- 快速浏览一遍，接受明显的优质图像
- 拒绝明显的劣质图像 (spill_rate > 0.7)
- 然后仔细审查边界案例

### 3. 利用筛选器
- 使用"spill_rate < 0.3"筛选器优先浏览高质量候选
- 使用"已接受"筛选器复查已选图像

### 4. 预期结果
- 8960张图像
- 预计筛选出 **150-200张** 高质量图像
- 筛选率约 **1.7-2.2%**

## 故障排除

### 端口被占用
```bash
# 使用其他端口
python filter_synthetic_images.py --data_dir /path/to/data --port 5001
```

### 图像显示不出来
- 检查`--data_dir`路径是否正确
- 确保目录下有`images/`文件夹

### 筛选决定丢失
- 筛选决定保存在浏览器localStorage中
- 清除浏览器缓存会导致决定丢失
- 定期导出结果作为备份

## 命令行参数

```bash
python filter_synthetic_images.py [OPTIONS]

Options:
  --data_dir PATH  批量生成输出目录 [必需]
  --host IP       Web服务主机 [默认: 127.0.0.1]
  --port PORT     Web服务端口 [默认: 5000]
```
