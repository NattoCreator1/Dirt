### 关于监督标尺版本的一致性说明

本文的严重度标尺由三个互补维度构成：不透明占比 (S_op)、空间分布 (S_sp) 与主导性 (S_dom)。在本稳健性分析中，我们采用其线性组合 **S = 0.6·S_op + 0.3·S_sp + 0.1·S_dom** 作为**代表性的三维标尺实例**，用于评估小规模数据训练在随机初始化与数据划分下的方差（multi-seed 与 k-fold）。

**需要强调的是**：
1. **稳健性分析的目标**是量化训练过程的随机性影响，而非为特定权重配置提供最优性证明
2. 该标尺版本属于"三维标尺族"的一个实例，我们已验证其与后续精炼版本在跨域排序趋势与总体结论上保持一致
3. 三维标尺体系相较"仅以脏污面积表征严重度"的简化方案在跨域排序相关性上显著更优，并具备完整消融记录支持

权重配置属于工程型超参，本文不以解析最优为目标；本文的核心贡献在于提出并验证"多维严重度标尺"相对单维面积标尺在跨域排序与可解释性上的优势，并进一步证明在小样本条件下训练过程的随机性不会主导该结论。

## 一、实验设计

### 1.1 研究问题

**核心问题**: WoodScape 训练集仅有 3,200 样本，baseline 性能是否受随机因素或数据划分影响？

**验证策略**:
1. **Multi-seed**: 测试对随机初始化的敏感性
2. **K-fold**: 测试对数据划分的敏感性

### 1.2 实验设置

| 参数 | Multi-seed | K-fold |
|------|-----------|---------|
| 变化维度 | 随机种子 (42, 123, 456) | 数据划分 (5 folds) |
| 训练集 | Train_Real (3,200) | Train+Val 重划分 (3,200/800) |
| 验证集 | Val_Real (800) | 每 fold 的验证集 (800) |
| 测试集 | Test_Ext (50,383 images, 505 clusters) | Test_Ext (50,383 images, 505 clusters) |
| Checkpoint | ckpt_last.pth (40 epochs) | ckpt_last.pth (40 epochs) |
| Bootstrap | 簇层面, 1000 次 | 簇层面, 1000 次 |

**监督标尺**: 本实验使用三维标尺的代表性实例 **S = 0.6·S_op + 0.3·S_sp + 0.1·S_dom**，其中：
- S_op: 不透明度感知的覆盖率
- S_sp: 空间加权的重要性
- S_dom: 主导性池化（含透明度折扣）

该配置作为"三维标尺族"的实例，用于验证训练过程的稳健性，而非声称权重最优性。

### 1.5 关键方法论修正

**⚠️ 5 项关键修正** (相比之前的分析):

1. **簇层面 Bootstrap**: 先聚合到 505 簇层面，再重采样簇（而非 50,383 图像）
2. **固定 Checkpoint**: 使用 ckpt_last.pth (固定 40 epochs) 而非 ckpt_best.pth
3. **K-fold 数据隔离**: Test_Real (1,000) 始终不参与 K-fold 训练/验证
4. **固定簇 ID 映射**: 所有 seeds/folds 使用相同的 cluster_id
5. **压缩比指标**: 新增对标尺压缩现象的量化评估

## 四、压缩比稳定性

### 4.0 标尺压缩比（compression ratio）的定义与解释

为了刻画模型在 External Test（弱等级标注）上的"等级响应幅度"是否被压缩，我们引入标尺压缩比指标。

**定义**:

具体地，先按外部等级 (level ∈ {1,2,3,4,5}) 计算预测严重度的分组均值：

```
m_ℓ = E[Ŝ | level = ℓ]
```

然后计算相邻 level 之间的预测差值：

```
Δ_ℓ = m_ℓ - m_{ℓ+1}
```

压缩比定义为所有相邻 level 差值的平均值：

```
compression_ratio = mean(Δ_5→4, Δ_4→3, Δ_3→2, Δ_2→1) / Δ_level_true
```

其中 Δ_level_true = 1（真实 level 差值）。

**解释**:

- 当 compression_ratio < 1 时，表示该模型对等级变化的敏感度低于理想情况（等级间隔被压缩）
- 当 compression_ratio 明显小于 1 时，通常对应"预测范围变窄 / 等级分离度降低"的现象
- 本报告中观察到的 compression_ratio ≈ 0.08，意味着模型对单位 level 变化的响应仅为真实变化的 ~8%

**需要注意**的是，该指标基于外部数据的弱等级标注，因此其物理含义是"相对等级响应幅度"，不应被解读为严格的绝对标尺误差比例；它更适合作为诊断信号，用于识别模型是否出现了系统性的标尺压缩趋势。

3. **双重验证**: Multi-seed + K-fold 全面评估稳健性
4. **可复现性**: 所有随机种子、数据划分、实验参数均明确记录
