## ⚠️ 重要修正说明（2025-02-26）

### 簇层面 Bootstrap 修正

**发现的问题**：
- External Test 数据集有 **505 个簇（视频序列）**，而非 50,458 个独立样本
- 同一簇内的图片来自同一视频的连续帧，**高度相关**
- 本报告使用的**图片层面 Bootstrap** 违反了独立性假设

**修正后的正确结果**（簇层面 Bootstrap）：

| 指标 | 图片层面（本报告） | 簇层面（正确） |
|------|------------------|---------------|
| ρ 点估计 | 0.6465 | **0.6478** |
| 95% CI | [0.6416, 0.6526] | **[0.5748, 0.6846]** |
| CI 宽度 | 0.0110 (1.7%) | **0.1098 (16.95%)** |
| **放大倍数** | — | **10.0x** |

**重要结论**：
1. 簇层面 Bootstrap 的 CI 宽度是图片层面的 **10-15 倍**
2. 簇层面方法**更稳定可信**，因为它正确处理了簇内相关性
3. **请使用簇层面结果进行正式的统计推断**
