## ⚠️ 重要修正说明（2025-02-26）

### 簇层面 Bootstrap 修正

**发现的问题**：
- External Test 数据集有 **505 个簇（视频序列）**，而非 50,458 个独立样本
- 同一簇内的图片来自同一视频的连续帧，**高度相关**
- 本报告使用的**图片层面 Bootstrap** 违反了独立性假设

**修正后的正确结果**（簇层面配对 Bootstrap）：

| 指标 | 图片层面（本报告） | 簇层面（正确） |
|------|------------------|---------------|
| Baseline ρ | 0.7177 | **0.7446** |
| Tile-only ρ | 0.6776 | **0.6846** |
| Δρ (Tile-only - Baseline) | -0.0401 | **-0.0600** |
| Δρ 95% CI | [-0.0482, -0.0354] | **[-0.0928, -0.0258]** |
| Δρ CI 宽度 | 0.0128 | **0.0670** |
| **放大倍数** | — | **5.2x** |

**重要结论**：
1. 簇层面 Bootstrap 的 Δρ CI 宽度是图片层面的 **5 倍**
2. 簇层面方法**更稳定可信**，因为它正确处理了簇内相关性
3. **请使用簇层面结果进行正式的统计推断**
